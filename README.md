# NorthStar AI - Frontend

## Overview

NorthStar AI Frontend is a modern web application built with React, Vite, and TypeScript. It provides an interactive user interface for authentication, dashboard, goal management, AI chat, and recommendations.

---

## Features

- User Login & Registration
- Dashboard
- Goal Tracking
- AI Chat Interface
- Recommendation Feed
- Responsive UI

---

## Tech Stack

- React
- Vite
- TypeScript
- CSS
